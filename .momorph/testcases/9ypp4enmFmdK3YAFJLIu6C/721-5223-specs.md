## 1. Screen Overview

The "Dropdown Profile" is a small dropdown overlay triggered by clicking the user avatar in the Header. It provides two menu items for regular (non-admin) authenticated users: "Profile" (navigate to user profile page) and "Logout" (sign out and redirect to login). The dropdown uses a dark background (#00070C) with gold-accent border (#998C5F), gold-tinted active states, and white text (Montserrat 16px Bold).

## 2. Design Items

| No | Item | Type | Description |
|----|------|------|-------------|
| A | Dropdown-List | others (dropdown_menu) | Dropdown container with dark background, gold border, 8px border-radius, padding 6px. Contains two menu items. Positioned absolutely (z-index 50). |
| A.1 | Profile | button (icon_text) | Menu item with user icon (24x24px) and label "Profile". Active state: gold-tinted background (rgba(255,234,158,0.10)), gold glow text shadow. Size: 119x56px, padding 0 16px. Click navigates to profile page. |
| A.2 | Logout | button (icon_text) | Menu item with chevron-right icon (24x24px) and label "Logout". Default state: transparent background, white text, no glow. Size: 121x56px, padding 0 16px. Click triggers Supabase signOut() and redirects to /login. |

## 3. Navigation Flow

- **Trigger:** Click user avatar in the Header (all authenticated pages)
- **Profile click:** Navigate to /profile, close dropdown
- **Logout click:** Call Supabase signOut(), redirect to /login
- **Click outside / Escape:** Close dropdown without action

## 5. Animations

- Dropdown open: opacity + translateY(-4px to 0), 150ms ease-out
- Dropdown close: opacity + translateY(0 to -4px), 100ms ease-in
- Item hover: background-color transition, 150ms ease-in-out

## 6. Accessibility

- Container: role="menu"
- Items: role="menuitem"
- Avatar trigger: aria-expanded
- Keyboard: Enter/Space to open, ArrowUp/Down to navigate items, Enter to activate, Escape to close

## 7. Functional Requirements

- FR-001: Display dropdown with "Profile" and "Logout" when avatar clicked
- FR-002: "Logout" calls Supabase signOut() and redirects to /login
- FR-003: "Profile" navigates to profile page
- FR-004: Dropdown closes on item selection, outside click, or Escape key

## 8. Edge Cases

- Click outside dropdown -> Close without action
- Session already expired when clicking Logout -> Redirect to /login
- Mobile viewport -> Dropdown positioned to avoid viewport overflow
- signOut() failure -> Display error notification, do not leave ambiguous auth state
