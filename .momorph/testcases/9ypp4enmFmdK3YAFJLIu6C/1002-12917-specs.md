## 1. Screen Overview

The "Addlink Box" is a modal dialog that allows users to add a hyperlink by entering display text and a URL. The modal contains a title, two input fields (Text and Link/URL), and two action buttons (Cancel and Save). The modal appears over a dimmed background.

## 2. Design Items

| No | Item | Type | Description |

| A | Title | label | Static title "Them duong dan" (Add link) displayed at the top of the modal |

| B.1 | Label "Text" | label | Label "Noi dung" to the left of the text input |

## 3. Validation Rules

### Text field (B / B.2):
- **Required:** Yes
- **Data type:** string
- **Min length:** 1 character
- **Max length:** 100 characters
- **Additional:** Must not contain only whitespace

### Link field (C / C.2):
- **Required:** Yes
- **Data type:** string
- **Format:** Valid URL (http/https)
- **Min length:** 5 characters
- **Max length:** 2048 characters

## 4. Navigation / Actions

- **Cancel button (D.1):** Closes modal, discards all changes
- **Save button (D.2):** Validates fields; if valid, saves link and closes modal; if invalid, displays error messages
- **Blur on Link field:** Validates URL format and shows error if invalid

## 5. Visual Specifications

- Modal background: Light cream/yellow
- Title: Bold, dark text, top-left aligned
- Input fields: White background, thin border, rounded corners, ~672x56px
- Cancel button: Small, outlined/bordered style with "X" icon
- Save button: Large, yellow background with link icon
- Labels: Left-aligned next to input fields ("Noi dung", "URL")
