# Specs: Dropdown-ngon-ngu (Language Dropdown)

## Screen Overview
- **Frame:** Dropdown-ngon-ngu (721:4942)
- **Status:** spec
- **Purpose:** Language selection dropdown component allowing users to switch between Vietnamese (VN) and English (EN).
- **Component Type:** Dropdown (not a standalone page - no dedicated URL)

## UI Elements

### A. Dropdown-List (ID: 525:11713)
- **Type:** Dropdown (others)
- **Description:** Dropdown for language selection.
- **Display:**
  - Selected area: top section, dark gray background, displays current language flag + language code
  - Option area: bottom section, dark/black background, displays alternate language flag + language code
- **Behavior:**
  - Click: opens/closes the dropdown menu
  - Select 'EN'/'VN': updates the displayed value and closes the menu
- **Size:** Container for language options

### A.1 Tieng Viet - Vietnamese (ID: I525:11713;362:6085)
- **Type:** List item
- **Description:** Language option for Vietnamese ('VN')
- **Display:**
  - Flag: Vietnam flag icon (left side)
  - Code: 'VN' text (white)
- **Behavior:**
  - Click: selects Vietnamese language and closes dropdown
  - States: hover shows highlight, selected has different background for distinction
- **Default:** VN is the default selected language

### A.2 Tieng Anh - English (ID: I525:11713;362:6128)
- **Type:** Button (icon_text)
- **Description:** Language option for English ('EN')
- **Display:**
  - Icon: UK/British flag icon (left side)
  - Label: 'EN' (white text)
  - Size: 110x56px, dark background
- **Behavior:**
  - Click: selects English language and closes dropdown, updates UI language
  - Hover: background changes to show interactive state
  - States: active shows highlight background

## Visual Design
- Dark theme: dark gray selected area, black/darker option area
- Flags displayed as small rectangular icons
- Text in white for contrast against dark backgrounds
- Compact layout suitable for header placement

## Functional Requirements
- Only 2 language options: VN (Vietnamese) and EN (English)
- Single selection only (one language active at a time)
- Selecting a language updates the entire application UI language
- Dropdown closes after selection
- Default language: VN (Vietnamese)

## Security Considerations
- No direct URL (component embedded in pages)
- Language preference should persist across sessions (cookie/localStorage)
- Language switching should not affect authentication state
