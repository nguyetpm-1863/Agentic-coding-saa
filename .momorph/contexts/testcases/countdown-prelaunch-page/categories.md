# Test Categories - Countdown - Prelaunch Page

## 1. ACCESSING

Test cases related to accessing and navigating to the Countdown - Prelaunch page.

| Sub Category | Description |
|---|---|
| Direct URL Access | Accessing the countdown page via direct URL |
| Browser Navigation | Using browser back/forward/refresh to access the page |
| Authentication | Verifying the page is accessible without login (public page) |
| URL Manipulation | Testing behavior with modified/invalid URLs |

## 2. GUI

Test cases related to the visual presentation and layout of the countdown page.

| Sub Category | Description |
|---|---|
| Layout | Overall page layout, element positioning, alignment |
| Days Counter Display | Visual display of the Days countdown unit |
| Hours Counter Display | Visual display of the Hours countdown unit |
| Minutes Counter Display | Visual display of the Minutes countdown unit |
| Labels | Display of "DAYS", "HOURS", "MINUTES" labels |
| Responsive Design | Layout behavior across different screen sizes and devices |
| Cross-Browser | Visual consistency across different browsers |

## 3. FUNCTION

Test cases related to the functional behavior of the countdown timer.

| Sub Category | Description |
|---|---|
| Countdown Calculation | Correct calculation of remaining days, hours, minutes |
| Auto-Update | Real-time countdown updates without user interaction |
| Boundary Values | Behavior at value boundaries (00, 23, 59, rollover) |
| Target Date Reached | Behavior when countdown reaches zero |
| Time Zone | Countdown accuracy across different time zones |
| Page Lifecycle | Countdown behavior on refresh, tab switch, background |
