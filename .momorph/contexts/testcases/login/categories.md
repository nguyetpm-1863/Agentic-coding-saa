# Login Screen - Test Case Categories

## 1. ACCESSING

Test cases related to access control, authentication state, and security of the Login screen.

| Sub_Category | Description |
|-------------|-------------|
| URL Access | Direct URL navigation to the login page |
| Authenticated Redirect | Behavior when already-authenticated users access the login page |
| Session Expiry | Behavior when session/token expires |
| Security Headers | HTTP security headers verification |

## 2. GUI

Test cases related to the visual appearance, layout, and UI elements of the Login screen.

| Sub_Category | Description |
|-------------|-------------|
| Screen Layout | Overall page layout, responsive design, element positioning |
| Header | Logo display, language selector appearance |
| Hero Section | Key visual, title, description text display |
| Login Button | Button appearance, states (default, hover, loading, disabled) |
| Footer | Copyright text display and positioning |

## 3. FUNCTION

Test cases related to the functional behavior of interactive elements on the Login screen.

| Sub_Category | Description |
|-------------|-------------|
| Google Login | Google OAuth authentication flow |
| Language Selector | Language switching functionality |
| Session Management | Token handling, auto-login, cookie behavior |
| Error Handling | Failed login, network errors, account issues |
| Multi-device | Login behavior across multiple devices/browsers |
