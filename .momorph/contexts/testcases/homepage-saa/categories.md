# Homepage SAA - Test Case Categories

## 1. ACCESSING

Test cases covering access control, authentication, and authorization for the Homepage SAA screen.

| Sub_Category | Description |
|-------------|-------------|
| Authentication | Verify page access based on login/logout state |
| Authorization | Verify feature visibility based on user roles (admin vs regular user) |
| URL Access | Verify direct URL access and redirection behavior |
| Session | Verify session handling and token refresh behavior |

Test cases covering the graphical user interface, layout, and visual presentation.

| Sub_Category | Description |
|-------------|-------------|
| Page Layout | Overall page structure, responsive breakpoints (320px, 768px, 1024px, 1440px) |
| Header | Header navigation bar display, logo, links, icons |
| Hero Section | Keyvisual banner, countdown timer, event info, CTA buttons |
| Awards Section | Section header, award card grid layout |
| Award Cards | Individual card display: thumbnail, title, description, detail link |
| Kudos Section | Sun* Kudos promotion block display |
| Footer | Footer navigation, logo, links, copyright text |
| Widget | Floating widget button display |
| Text & Labels | Static text content, fonts, colors, truncation |
| Hover States | Button/link/card hover visual feedback |
| Responsive | Mobile-first breakpoint adaptations |
